# markdown-highlight-mark

为 Obsidian 的高亮语法 `==...==` 提供支持，包括 markdown 文件和 Jupyter Notebook 的预览。

![](example/test_md.png)

![](example/test_ipynb.png)

## 使用方法

### 基本使用

基本: ==这是高亮==

嵌套: ==**粗体高亮**==

### 快捷键

| 快捷键             | 功能                             |
| ------------------ | -------------------------------- |
| `Ctrl + Shift + =` | 包装选中的同一行文本为 `==...==` |

> 注意：快捷键可能会与其他扩展或 vscode 功能冲突，请根据需要调整快捷键设置。

### 右键菜单

- 右键选择 "高亮选中文本" 包装选中文本
